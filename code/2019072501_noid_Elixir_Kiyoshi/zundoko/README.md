# ズンドコキヨシ feat. Elixir

いわゆるズンドコキヨシ。独自拡張仕様有り。

## Specification

1.  「ズン」「ドコ」をランダムに出力し続ける。
2.  「ズン」「ズン」「ズン」「ズン」「ドコ」が並んだ場合、「キ・ヨ・シ！！！！」を出力して終了する。
3.  但し、「ズン」「ズン」「ズン」「ズン」「ズン」「ドコ」はオーバーズンとし、キヨシと認めない。
4.  「ドコ」「ドコ」「ドコ」「ドコ」「ドコ」が並んだ場合、特別に「ヨ・シ・キ！！！！」を出力して終了する。

## How to run

```shell
mix run -e "Zundoko.main()"
```

## How to test

```shell
mix test
```
